# allinone
## Working with VPS
### connect via ssh
```
root@ip_address
```

### upload file to vps
```
scp /path/to/local-file root@ip_address:/remote/path/to/upload/to
```

### download file from vps
```
scp root@ip_address:/path/to/remote/file /local/path/to/download/to
```
